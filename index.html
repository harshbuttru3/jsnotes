Chapter 1

variables and Data

Just like we follow some rules while speaking english (the grammar), we have some rules to follow while writing a javascipt program. The set of these rules is called syntax in javascipt.

What is a variable?

A variable is a container that stores a value. This is very similar to the containters used to store rice, water and oats (Treat this as analogy).

The value of a javascipt variable can be changed during the execution of the program.

 
        var a = 7;         
        let a = 7;        
        const a = 7;

Rules for changing variable names

  • Letters, digits, underscores & $ sign allowed
  • Must begin with a $, _ or a letter
  • javascipt reserved words cannot be used as a varible name
  • Harsh and haRsh are different varibles (case sensitive)

var vs let in javascipt

  1. "var" is globally scoped while "let" and "const" are block scoped.
  2. "var" can be updated & re-declared within its scope.
  3. "let" can be updated but not be re-declared.
  4. "const" can neither be updated nor be re-declared
  5. var variables are initialized with undefined whereas let and const variables are not initialized
  6. const must be initialized during declaration unlike let and var

Conditional statements

In javascipt we have three forms of coditional statements

  1. if statement
  2. if... else statement
  3. if ... else if ... else statement

  1. if statement

    2. The if statement in javascipt looks like this:-

        
      if (condition) {                            
                      //executes the code         
      }

    The if statement evaluate the condition inside the (). If the condition is evaluated to true, the code inside the body of "if" is executed else the block of code is skipped

  2. if-else statement

    3. Sometimes, we might want to keep rechecking a set of conditions one by one until one matches.
    syntax of if...else statement looks like this

        
      if (age > 0) {                             
                     console.log("A valid age"); 
                   }                             
      else {                                     
        console.log("This is invalid age");      
      }

  3. if...else-if...else statement

    4. syntax of if...else statement looks like this.

        
      if (age > 0) {                        
        console.log("A valid age");         
      }                                     
      else if (age > 10 && age < 15) {      
        console.log("but you are a kid");   
      }                                     
      else if (age > 15 ) {                 
        console.log("Not a kid");           
      }                                     
      else {                                
        console.log("invalid age")          
      }

Javascript ternary operator

evaluate a condition and execute a block of code based on the condition.
syntax:- condition? expression1: expression2 . for example:- 

  
    let totalMarks = 100;                                                         
    let marks = 45;                                                               
    (marks > 30) ? console.log("You are passed"): console.log("You are failed")

"You are passed" will be printed in console.

Chapter 3

We use loops to perform repeated actions

The for loop

  
    for (statement1; statement2; statement3) {    
      //code to be executed                       
    }
  • statement 1 is executed one time
  • statement 2 is the condition based on which the loop runs.(loop body is executed)
  • statement 3 is executed everytime the loop body is executed

The for-in loop

  
    for(key in object) {                        
                        //code to be executed   
    }

Note:- For-in loops also works, with arrays which will be discussed later

The for-of loop

  
    for (variable of iterable) {                      
                                //code to be executed 
    }

iterable means iterable data sturcture like arrays, strings, etc

While loop

  
    while (condition) {                   
                  //code to be executed   
    }

If the condition never becomes false, the code never stop and may crash the run time.

The do while loop

The do while loop's syntax looks like this:-

  
    do { //code to be executed }    
        while (condition)
